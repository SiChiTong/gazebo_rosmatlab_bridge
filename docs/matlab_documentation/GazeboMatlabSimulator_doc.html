GazeboMatlabSimulator
 Matlab client which handles the communication with Gazebo.
 Author Gowtham Garimella (ggarime1@jhu.edu)
Class Details
Superclasses handle
Sealed false
Construct on load false
Constructor Summary
GazeboMatlabSimulator Starts Matlab Gazebo bridge and provides available links and joints 
Property Summary
ActuatedJoints Index of joints being actuated. (The index starts with 1 and is based on available names provided above)  
ActuatedLinks Index links being actuated. (The index is based on Availableindices) 
AvailableNames Helper Property which provides the available names to use. The first cell provides links available, second cell provides joints available 
MexData Stored data for mex (Internal) 
PhysicsTimeStep Time step for running physics engine should be configured using Configure function 
Method Summary
  AttachServo Attaches a servo to given joint indices 
  Configure the physics engine to run with time step  
  PublishTrajectory Publish the trajectory: 
  Reset the Gazebo world 
  SetModelState Set the Model state(Pose) and joint angles of the joints 
  Step Run a single step of size (stepsize) of gazebo with given controls 
  StepTrajectory This runs multiple steps at once and provides the link and joint states at all the times ts
